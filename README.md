# React Profile Cards App

This is a simple React project built to understand and implement:
- React props
- Dynamic rendering using map()
- Reusable components
- Basic Flexbox layout

## Tech Stack
- React (Vite)
- JavaScript (ES6)
- CSS

## Features
- Profile cards generated dynamically from an array
- Reusable Card component
- Clean and responsive UI

## Learning Outcome
- Understood how props work in React
- Learned to pass data from parent to child components
- Used map() to render multiple components dynamically
